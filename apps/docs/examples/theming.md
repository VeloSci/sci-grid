# Theming

Each grid configuration can be heavily customized to match your application's theme, whether it is Light, Dark, or Cyberpunk Green.

```typescript
const config = {
  backgroundColor: '#000000',
  gridLineColor: '#003b00',
  textColor: '#00ff41',
  headerBackground: '#001a00',
  headerTextColor: '#00ff41',
  font: '14px "Courier New", monospace',
  selectionColor: 'rgba(0, 255, 65, 0.2)',
  selectedTextColor: '#000'
};
```
