Introduction

SciGrid is a high-performance, canvas-based data grid for web applications. It is designed to handle millions of rows with smooth scrolling (60fps) and low memory footprint.

Unlike DOM-based grids, SciGrid uses a virtualization engine and HTML5 Canvas to render only what is visible, enabling it to scale comfortably to large datasets.

Installation

Install SciGrid via npm or pnpm:

npm install sci-grid

Or using pnpm:

pnpm add sci-grid

Quick Start

Get up and running with a basic grid in seconds.

import { SciGrid } from 'sci-grid';

const container = document.getElementById('grid');
const provider = {
    getRowCount: () => 1000,
    getColumnCount: () => 5,
    getCellData: (r, c) => 'Cell ' + r + ':' + c,
    getHeader: (c) => ({ name: 'Col ' + c })
};

const grid = new SciGrid(container, provider);

Column Types

SciGrid supports multiple column types to visualize data effectively.

Interactive Demo

Supported Types

  • text: Standard text display (default).
  • numeric: Right-aligned numbers.
  • checkbox: Interactive boolean toggle.
  • progress: Visual percentage bar (0-100).
  • select: Dropdown list for fixed options.

Theming
